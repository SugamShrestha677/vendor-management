# Vendor & Procurement Workflow Management System

A comprehensive MERN stack application for managing vendor-related procurement workflows with role-based access control (Employee, Manager, Vendor).

## 🎯 Features

### Core Functionality
- **Purchase Request Management**: Employees create and track purchase requests
- **Approval Workflow**: Managers review and approve/reject requests
- **Vendor Management**: Manage vendor details and ratings
- **Purchase Order Tracking**: Monitor order status and delivery
- **Real-time Notifications**: Instant updates on request status changes
- **Advanced Reporting**: Generate analytics and export reports
- **Search & Filtering**: Quick access to requests, vendors, and orders

### User Roles
1. **Employee**: Create purchase requests, track status
2. **Manager**: Review and approve requests, manage vendors
3. **Vendor**: Receive orders, update delivery status

### Security Features
- JWT-based authentication
- Password hashing with bcrypt
- Input validation and sanitization
- CORS protection
- Rate limiting
- Activity logging
- Role-based access control (RBAC)

### Dashboard Features
- Role-specific analytics
- Real-time statistics
- Visual charts and graphs
- Recent activity tracking
- Budget monitoring

## 📋 Prerequisites

Before you begin, ensure you have installed:
- Node.js (v16 or higher)
- MongoDB (local or Atlas)
- npm or yarn
